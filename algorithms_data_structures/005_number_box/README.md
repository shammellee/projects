# Number Box

This program prints a box of numbers whose size is derived from user input.

* Dependencies
  * Make
  * CMake
  * GCC or Clang

## Building and Running Number Box

```sh
$ cd algorithms_data_structures/005_number_box
$ mkdir build
$ cd build
$ cmake ..
$ make
$ ./bin/number_box
```
